##编译安装方法
------------
	./configure  
	make  
	[sudo] make install  


##使用方法
-----------
bash:~# nali **IP**  
即可得到IP所在地区的信息  
采用的纯真IP数据库  
nali-update更新数据库  
(<https://github.com/out0fmemory/qqwry.dat> 此项目为自动更新纯真数据库)  
nali-ping 直接ping域名对应IP的所在地区
